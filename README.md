# Skills

A collection of reusable agent skills for AI coding assistants.

## Available Skills

| Skill | Description |
| --- | --- |
| [design-an-interface](design-an-interface/) | Generate multiple radically different interface designs using parallel sub-agents, then compare trade-offs. Based on "Design It Twice". |
| [prd-to-issues](prd-to-issues/) | Break a PRD into independently-grabbable GitHub issues using vertical slices (tracer bullets). |
| [prompt-review](prompt-review/) | Review and optimize prompts for LLM reasoning models. Checks clarity, structure, anti-patterns, and agentic concerns. |
| [skill-authoring](skill-authoring/) | Create and review agent skills with proper structure, progressive disclosure, and bundled resources. |
| [tdd](tdd/) | Test-driven development with red-green-refactor. Vertical slices, integration-style tests, behavior over implementation. |
| [write-a-prd](write-a-prd/) | Interview the user, explore the codebase, then write a PRD as a GitHub issue with user stories, implementation and testing decisions. |
| [writing-coach](writing-coach/) | Review, rewrite, or draft articles matching Johann's personal writing style. |
